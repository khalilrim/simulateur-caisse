## Application Simulateur de Caisse

## Description

Cette application web permet de simuler une caisse de magasin.
L’utilisateur peut ajouter des produits, calculer automatiquement les montants et générer un ticket.

## Fonctionnalités

-Ajouter un produit au panier
-Supprimer un produit
-Calcul automatique du sous-total
-Calcul automatique des remises
-Génération d’un ticket de caisse
-Vider le panier
-Sauvegarde avec LocalStorage

## Technologies utilisées

-HTML5
-CSS3
-JavaScript natif
